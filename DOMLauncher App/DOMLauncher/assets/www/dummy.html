DOMLauncher

Well hello there!

You are seeing me because of a few possibilities. First possility is that your external sdcard is not accessable. DOMLauncher asset files are are on the internal sdcard but not all phones have this feature. If DOMLauncher can not default to the internal the external will be used. If the external sdcard is mounted or otherwise unreadable when DOMLauncher loads, this 'dummy' view will appear. Second posibility is that the wrong folder name is typed within the 'activeTheme.txt' file within the 'settings' folder. The last posibility is that the chosen theme does not have an index.html file.

Icon Cache

The Icon Cache is the folder 'icons' in the DOMLauncher folder. This will only work if your SD CARD (either internal or external) is accessible. Here all installed applications have their icon exported in the png format. There is also an auto-generated css file holding the standard DOMLauncher references for easy icon placement within the UI Plugins.

Exit App

So the app does not get stuck in a loop, and incase a device does not have a 'back' button, tap this button to exit. Hit your home button to relauch DOMLauncher afterwards.
